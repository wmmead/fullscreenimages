Fixed Width Large Background Images

Above, we have a page set to a fixed width. The image in the header is in the background, and since the image is twice as wide as it is tall, you can easily set the height of that header to half the width of the picture. Notice that the image is cropped. This is because the actuall photo is 2000 pixels wide and does not fit. This can be changed with the background size property. Set to contain, the entire background image will be displayed in the container. This works here because the proportions of the header container are exactly the same proportions of the image.

However, this is clearly not a "full screen" image, since a screen could be wider or narrower than 1200px. When we get to the flexible screen sizes, we run into problems with the specifically defined widths above.
